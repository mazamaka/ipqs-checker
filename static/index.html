Fingerprint Checker

Проверка антидетект-браузеров через IPQS, Fingerprint Pro и CreepJS

IPQS via indeed.com
  • Fraud Score (0-100%)
  • OS Mismatch детект
  • VPN/Proxy/Tor детект
  • Canvas/WebGL хеши
FP Pro via fingerprint.com
  • Anti-detect браузер детект
  • Suspect Score
  • Bot Detection
  • Tampering детект
CreepJS open-source
  • Headless детект
  • Stealth Score
  • Lie/Trash детект
  • Resistance анализ
1 Установите расширение
Скачайте и установите для Chrome
2 Выберите сервис и проверьте
В popup выберите IPQS, Fingerprint Pro или CreepJS и нажмите "Проверить"
3 Получите детальный отчёт
Откроется страница с полным анализом вашего fingerprint
